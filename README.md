# LostAndFind
失物招领平台为个人项目，主要是为了检验学习了一段时间后对前端知识的掌握程度，顺便夯实基础，不作它用。		
因此项目中会引用到别的网站的素材(比如 [失物认领](http://www.shiwurenling.com/ "参考网站")),如若不妥，请联删，谢谢。
